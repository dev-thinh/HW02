# TC-MPROD-BVA-002: Thêm vào giỏ hàng với số lượng bằng 1 (MIN)

## Requirement ID

FR-06 (Mobile)

## Module / Test type / Technique

Mobile-Product / Functional / BVA (quantity MIN)

## Preconditions

- Đang ở màn hình chi tiết sản phẩm `iPhone 15 Pro Max` trên Mobile App.

## Test data

| Field    | Value |
| -------- | ----- |
| Số lượng | `"1"` |

## Test steps

1. Nhập số lượng `"1"` vào ô số lượng.
2. Nhấn nút **Thêm vào giỏ hàng**.

## Expected result

- Thêm vào giỏ hàng thành công.
- Số lượng sản phẩm trong giỏ hàng tăng thêm 1.

## Status / Related bugs

Pass / None
